Get started

API Endpoint

https://1800.fyi/api/

The 1800.fyi API provides programmatic access to get rhymes for a particular word.

get rhymes


# Here is a curl example
curl \
-X GET https://1800.fyi/api/word/

To get rhymes from desired word you need to make a GET call to the following url :
https://1800.fyi/api/word/:word
